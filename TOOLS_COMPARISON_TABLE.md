## 📌 Quick Notes

### Getting GitHub Student Developer Pack
The GitHub Student Developer Pack unlocks free pro tiers on many tools above. To apply:
1. Go to [education.github.com/pack](https://education.github.com/pack)
2. Sign in with your GitHub account
3. Verify your student status (use your `.ac.id` university email)
4. Approval usually takes 1–7 days

**Student Pack highlights relevant to this workshop:**
- ✅ **GitHub Copilot** — Free (normally $10/mo)
- ✅ **GitHub Codespaces** — Extra free hours
- ✅ **Vercel Pro** — Free (normally $20/mo)
- ✅ **Replit Hacker** — Free (normally $7/mo)
- ✅ **Namecheap domain** — Free .me domain for 1 year

---

### Works Offline Notes
Most AI tools require internet for AI features. However:
- **VS Code** editor functionality is fully offline
- **Git** CLI works offline (push/pull require internet)
- **Continue.dev** can use local models (Ollama) for offline AI
- **Aider** can use local models via Ollama for offline AI
- **Semgrep CLI** can run scans offline with pre-downloaded rules

### Choosing the Right Tool for the Task

| Task | Recommended Tool(s) |
|------|---------------------|
| Writing code from scratch | Claude Web, ChatGPT, Cursor, Claude Code |
| Editing code in your editor | GitHub Copilot, Cursor, Windsurf, Continue.dev |
| Automating terminal tasks | Claude Code, Aider |
| Building full apps fast | Lovable, Replit |
| Automated PR review | CodeRabbit |
| Security scanning | Semgrep, GitGuardian, GitHub Secret Scanning |
| Deploying a static site | GitHub Pages, Netlify, Vercel |
| Learning / exploring a codebase | Macroscope, Claude Code `/init` |
