Claude Desktop — Local Power

A native app that gives Claude access to your local files, GitHub, and tools via MCP servers.

Install Claude Desktop

🔗
Download

claude.ai/download — Windows & Mac

After installing, sign in with your Anthropic account. The MCP configuration file is at:

# Mac
~/Library/Application Support/Claude/claude_desktop_config.json

# Windows
%APPDATA%\Claude\claude_desktop_config.json

What is MCP?

Model Context Protocol is an open standard that lets Claude connect to external tools. Think of it as a universal adapter that gives Claude real access to your computer.

🗂️

Filesystem MCP

Read/write local files directly

↓
🐙

GitHub MCP

Issues, PRs, commits via API

↓
🌐

Browser MCP

Navigate, screenshot, extract

Connect the Filesystem MCP

npm install -g @modelcontextprotocol/server-filesystem
// claude_desktop_config.json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "/Users/yourname/projects"
      ]
    }
  }
}

Claude Code — Terminal Agent

An autonomous coding agent that lives in your terminal, reads your entire codebase, and makes changes directly.

⚠️
Always review every AI-generated change before committing to Git

Run git diff before every git add. You are responsible for the code in your repo.

Install Claude Code

# Requires Node.js 18+
npm install -g @anthropic-ai/claude-code

# Verify
claude --version

# Authenticate
claude auth login

# Run in any project
cd ~/projects/my-app && claude

MCP Dispatch — Full Automation

Route tasks across multiple MCP servers — filesystem, GitHub, browser — in a single natural language instruction.

💡
What "Dispatch" Means

When you give Claude a natural language instruction, it decides which MCP server(s) to call, executes them in sequence, and integrates the results — all invisibly to you.

Available MCP Servers

🗂️ Filesystem

npm install -g @modelcontextprotocol/server-filesystem

Read/write files, list directories, move/copy files

🐙 GitHub

npm install -g @modelcontextprotocol/server-github

Repos, issues, PRs, commits — requires PAT token

🌐 Browser / Puppeteer

npm install -g @modelcontextprotocol/server-puppeteer

Navigate URLs, screenshot, extract content

💬 Slack

npm install -g @modelcontextprotocol/server-slack

Read/send messages, list channels — requires bot token

Example Dispatch Workflow

Single Prompt — Multi-Server Execution 
"Check my GitHub notifications for new PR reviews.
For each PR, read the changed files locally.
Write a daily standup summary and save it to
~/daily-standup-2025-01-15.md"
GitHub MCP → list PRs Filesystem MCP → read files Filesystem MCP → write summary

Security Rules for MCP

🔐
Least Privilege Always
  • Only grant filesystem access to your projects folder — not /
  • Use GitHub tokens with minimum required scopes
  • Review every file operation before confirming
  • Never store tokens directly in config files that get committed
