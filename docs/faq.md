# FAQ — Padang AI Coding Tools Class

**Q: Do I need to pay for any tools in this workshop?**
No. Every tool used in this workshop has a meaningful free tier. The only optional upgrade is GitHub Copilot — which is free for students via the GitHub Student Developer Pack.

**Q: I don't know how to code. Can I still attend?**
Yes. The workshop is designed for beginners. Basic computer literacy (using a browser, creating files) is sufficient. Prior coding knowledge makes some sections easier, but is not required.

**Q: What laptop do I need?**
Any laptop from the last 5 years running Windows, Mac, or Linux. You need a modern browser (Chrome/Firefox), internet access, and ~2GB free disk space for tools.

**Q: Can I use my phone instead of a laptop?**
For most activities, no — coding tools require a keyboard and proper screen. However, you can follow along and use Claude Web on mobile. For the hands-on exercises, a laptop is strongly recommended.

**Q: Is my work private when I use Claude or ChatGPT?**
By default, conversations may be used to improve AI models. Never paste passwords, API keys, personal data, or confidential business information into any AI chat window. For sensitive work, check the privacy settings of each tool.

**Q: Can AI write all my code for me?**
AI can generate code quickly, but you are responsible for reading, understanding, and testing it before committing or deploying. Blindly running AI-generated code is a significant source of bugs and security vulnerabilities.

**Q: What's the difference between Claude and ChatGPT?**
Both are large language model assistants. Claude (by Anthropic) has a larger context window (200k tokens vs 128k), tends to be more careful about acknowledging uncertainty, and has a unique code-in-terminal tool (Claude Code). ChatGPT (by OpenAI) has a larger user community and is slightly better at some explanation tasks for beginners. Use both — they complement each other.

**Q: My GitHub Copilot isn't working. What do I do?**
Check: (1) Is the extension installed in VS Code? (2) Are you signed in to GitHub? (3) Do you have an active Copilot subscription or Student Pack access? See the troubleshooting guide for step-by-step fixes.

**Q: How do I apply for the GitHub Student Developer Pack?**
Go to https://education.github.com/pack, sign in with your GitHub account, and verify your student status with your university email (`.ac.id` addresses are usually automatically approved) or student ID card photo. Approval takes 1–3 days.

**Q: Can I use AI tools for university assignments?**
This depends entirely on your institution's academic integrity policy. Always check with your instructor before using AI assistance on graded work. Using AI to *learn* is always fine; submitting AI-generated work as your own without disclosure may violate academic rules.

**Q: What should I do after the workshop?**
1. Apply for GitHub Student Developer Pack if you haven't yet
2. Deploy your first project to GitHub Pages
3. Add CodeRabbit to your class project repo
4. Build one of the five mini-projects in the Projects section
5. Join the workshop group chat for peer support
