## 1. GitHub

**What is free:** Everything you need for version control and collaboration.

GitHub's free tier includes unlimited public and private repositories, unlimited collaborators on public repos, GitHub Actions (2,000 CI/CD minutes/month), GitHub Pages (static site hosting), GitHub Issues and Projects (project management), and the GitHub mobile app. There is no time limit — the free tier is permanent.

**How to sign up:** Go to https://github.com and click "Sign up". Use your university email address — this is essential for accessing the Student Developer Pack and free GitHub Copilot. The sign-up process takes less than 5 minutes and requires no credit card.

**Limitations:** Private repos have some collaboration limits on the free tier. GitHub Actions minutes are limited to 2,000/month (resets monthly). Large File Storage (LFS) for binary files has costs above the free 1GB.

**Best free use case:** Host your entire portfolio of class projects as public repositories. GitHub Pages lets each project have its own live URL. Recruiters and employers routinely check GitHub profiles when evaluating candidates.

**Upgrade path:** GitHub Pro ($4/month) adds more Actions minutes, larger LFS storage, and additional repository insights. Most students never need to upgrade.

## 2. GitHub Student Developer Pack ⭐ FEATURED

**URL:** https://education.github.com/pack

**What is free:** Access to 100+ professional developer tools, all free for verified students.

The GitHub Student Developer Pack is the single most valuable free resource available to university students in technology fields. It is a curated collection of professional tools that normally cost significant money, offered for free through partnerships between GitHub and technology companies. The pack is specifically designed to give students access to the same tools used by professional developers.

**Notable inclusions as of this writing:**
- **GitHub Copilot** — AI coding assistant (normally $10/month = $120/year)
- **GitHub Codespaces** — extra compute hours for cloud development
- **Namecheap** — free `.me` domain name for 1 year (your portfolio site!)
- **DigitalOcean** — $200 in cloud computing credits
- **JetBrains** — free professional IDEs (IntelliJ, PyCharm, WebStorm) for 1 year
- **MongoDB Atlas** — $50 in cloud database credits
- **Heroku** — platform computing credits
- **Canva Pro** — free for 1 year (design your project presentations)
- **Microsoft Azure** — $100 in cloud credits
- **DataCamp** — free access to data science courses
- **Educative.io** — 6 months free for interactive coding courses

The pack is renewed as long as you remain a student. Apply every year to keep access.

**How to apply:**
1. Go to https://education.github.com/pack
2. Sign in with your GitHub account (use university email if possible)
3. Click "Get your pack" and select "Student"
4. Verify your student status using:
   - University email address (easiest — automatic approval for many universities)
   - Student ID card photo (upload a clear photo)
   - Enrollment verification letter
5. Wait 1-3 business days for approval
6. You'll receive an email confirmation

**Important for Indonesian students:** Many Indonesian universities (Universitas Padang, UI, ITB, ITS, BINUS, etc.) have email addresses that GitHub Education recognizes. If your university email doesn't work automatically, upload your student ID card (KTM) photo.

**Limitations:** Requires current student enrollment. Tools have their own terms — some are free for 1 year then discounted. The pack itself must be renewed each academic year.

**Best free use case:** Apply on your first week at university and keep renewing. The GitHub Copilot access alone ($120/year value) is worth the 10-minute application process many times over.

## 3. GitHub Copilot

**What is free:** Full GitHub Copilot access for verified students and educators (via Student Developer Pack).

GitHub Copilot is AI-powered code completion that integrates directly into VS Code, JetBrains IDEs, and other editors. It provides real-time inline code suggestions as you type, a chat panel for asking coding questions, slash commands (`/explain`, `/fix`, `/tests`), and code review capabilities. It's built on models from OpenAI and GitHub's own training.

For students who don't have the pack yet, GitHub also offers a limited free tier (as of 2024) with 2,000 code completions per month and 50 chat messages per month — enough to evaluate the tool before applying for the Student Pack.

**How to sign up:** Get the Student Developer Pack (see above) — GitHub Copilot is automatically included. Then in VS Code, install the "GitHub Copilot" and "GitHub Copilot Chat" extensions and sign in with your GitHub account.

**Limitations:** Suggestions can be confidently wrong. Quality varies by language popularity (Python/JavaScript are better than niche languages). The inline completions use your open files as context — it's not aware of your entire project by default without using Copilot Chat's `@workspace` feature.

**Best free use case:** Use Copilot daily for autocompletion while learning. Writing `# function to validate Indonesian phone number format` and pressing Tab teaches you both the implementation and the practice of writing descriptive code comments.

**Upgrade path:** GitHub Copilot Pro ($10/month) adds more chat messages. Individual plan is available for those without Student Pack access.

## 4. GitHub Codespaces

**What is free:** 60 hours/month on a 2-core cloud development environment, 15GB storage.

GitHub Codespaces launches a full Linux development environment in your browser (or connects to local VS Code). Each Codespace is a Docker container with VS Code, a terminal, and your repository's code — accessible from any device with a browser. The 60 free hours resets every month. Student Developer Pack recipients may receive additional hours.

**How to sign up:** No separate sign-up needed — Codespaces is part of GitHub. Open any repository and click the **Code** button, then the **Codespaces** tab, then **Create codespace on main**.

**Limitations:** Stops automatically after 30 minutes of inactivity (free tier). 60 hours/month is generous for casual use but may run out during intensive project work. Slower than local development for some workflows.

**Best free use case:** Use Codespaces for workshop exercises and collaborative projects. It eliminates "it works on my machine" problems because everyone runs the same environment. Also invaluable if your personal computer is low-spec.

**Upgrade path:** GitHub Team/Enterprise plans include more Codespaces hours. Individual additional hours can be purchased.

## 5. Claude.ai

**What is free:** Access to Claude 3.5 Sonnet (approximately 30 messages/day) and Claude 3 Haiku (more available).

Claude.ai's free tier provides access to powerful Claude models without a credit card. The 200,000-token context window — even on the free tier — is the largest available context window of any major AI assistant. This means Claude can read entire codebases, large PDF textbooks, or long documents in a single conversation. The Projects feature on the free tier allows you to create persistent AI workspaces.

The daily message limit on Sonnet (~30 messages) may feel restrictive for intensive use, but is very reasonable for studying and targeted help. When you reach the Sonnet limit, Haiku remains available and is still a capable model.

**How to sign up:** Go to https://claude.ai and sign up with Google, email, or Apple ID. No credit card required.

**Limitations:** Daily message limits on the most capable model (Sonnet). Claude Opus (the most powerful model) requires Pro subscription. No internet access within conversations on free tier.

**Best free use case:** Use Claude for deep dives — pasting in an entire file for code review, uploading a PDF assignment for explanation, or working through a complex algorithm step-by-step. The long context window is unique and very valuable for students.

**Upgrade path:** Claude Pro ($20/month) removes daily limits, adds Opus access, and gives priority during peak times.

## 6. ChatGPT

**What is free:** GPT-4o with daily usage limits; GPT-4o mini with no limits.

ChatGPT's free tier gives access to GPT-4o — one of the most capable AI models available. There are daily limits on GPT-4o usage, but GPT-4o mini is available without limits and is quite capable for many tasks. The interface is polished, the community is huge, and there are many shared prompt libraries and GPTs (custom assistants) available.

ChatGPT's Code Interpreter (now called Advanced Data Analysis) can run Python code, create charts, and process uploaded files — available on the free tier with limits.

**How to sign up:** Go to https://chatgpt.com and create a free account. No credit card required. Sign up with Google, Microsoft, Apple, or email.

**Limitations:** Context window is 128k tokens (smaller than Claude's 200k). Daily limits on GPT-4o. Some advanced features (custom GPTs creation, advanced voice) may require Plus.

**Best free use case:** Quick coding questions, concept explanations, and generating boilerplate code. ChatGPT is excellent at explaining code to beginners in clear, structured language. Also great for practicing algorithm explanations before a technical interview.

**Upgrade path:** ChatGPT Plus ($20/month) removes limits on GPT-4o, adds GPT-4o with voice, image generation with DALL-E, and priority access.

## 7. Cursor

**What is free:** Free Hobby plan with 2,000 code completions/month and limited chat messages.

Cursor is an AI-first code editor built on VS Code. The free Hobby plan includes 2,000 autocomplete suggestions per month, a limited number of AI chat uses, and the full VS Code experience with all extensions. It's an excellent way to experience AI-native development before committing to a paid plan.

For students, 2,000 completions/month is enough for moderate use. Many students find that being mindful about when to use completions (for boilerplate and repetitive code) makes the free tier last all month.

**How to sign up:** Go to https://cursor.sh, download the app, create a free account, and start coding. No credit card needed for the Hobby plan.

**Limitations:** 2,000 completions/month resets monthly. Limited chat uses. More advanced "Composer/Agent" features require Pro.

**Best free use case:** Use Cursor for class projects where you want AI-assisted coding but can't afford Copilot yet (before getting the Student Pack). The Hobby plan is great for learning how AI coding tools feel in practice.

**Upgrade path:** Cursor Pro ($20/month) provides unlimited completions and chat, plus the full Composer/Agent experience.

## 8. Google AI Studio / Gemini

**What is free:** Free API key with generous rate limits for Gemini 1.5 Flash and Gemini 1.5 Pro.

Google AI Studio (https://aistudio.google.com) provides a free API key for Google's Gemini models with no credit card required. Gemini 1.5 Flash offers 15 RPM (requests per minute) and 1 million tokens/minute free. Gemini 1.5 Pro offers 2 RPM free. These limits are generous for student projects and learning.

Critically, this free API can be used with tools like Continue.dev and Aider, giving you a free Claude/Copilot alternative for daily development. The 1-million-token context window on Gemini 1.5 Pro is the largest available context window in any AI model.

**How to sign up:** Go to https://aistudio.google.com, sign in with your Google account, and click "Get API Key". The key is created instantly with no billing setup required.

**Limitations:** Rate limits apply (requests per minute). For production apps, you'll need to set up billing. Gemini's coding quality varies compared to GPT-4o and Claude Sonnet, but is improving rapidly.

**Best free use case:** Connect the Gemini API to Continue.dev for free unlimited code completions in VS Code, or use with Aider for a free terminal coding assistant. Also excellent for building AI-powered apps (text analysis, summarization) in Python or JavaScript projects.

**Upgrade path:** Google AI Studio billing enables higher rate limits. Gemini Advanced ($20/month via Google One) provides access in the consumer interface with more features.

## 9. CodeRabbit

**What is free:** Full AI code review on all public repositories — completely free, forever.

CodeRabbit provides automated AI code review on GitHub Pull Requests. For public repositories, this is 100% free with no usage limits. Every PR you open in a public repo gets a full review: bug detection, security issue identification, style suggestions, and a plain-English summary. It installs in minutes and requires no code changes.

For students with public class project repositories, this is an extraordinary resource. It's like having a senior developer review every PR for free.

**How to sign up:** Go to https://coderabbit.ai, click "Install", and authorize the GitHub App. Select your repositories. That's it — no configuration needed to start getting reviews.

**Limitations:** Private repositories require a paid plan. Reviews can be verbose on large PRs. May produce false positives (you can dismiss these).

**Best free use case:** Install on your class project repo on Day 1. Let CodeRabbit catch the obvious issues before your instructor reviews. Reading CodeRabbit's explanations is an education in itself — it teaches you security and quality best practices through concrete examples.

**Upgrade path:** CodeRabbit Pro covers private repositories (pricing varies). Teams can configure reviews with `.coderabbit.yaml`.

## 10. Vercel

**What is free:** Hobby tier — unlimited personal projects, 100GB bandwidth/month, automatic HTTPS.

Vercel is the premier platform for deploying frontend applications. The free Hobby tier is genuinely unlimited for personal projects — there's no project limit, no deployment limit, and no time limit. Connect your GitHub repo and every push to main automatically deploys in under 60 seconds. Every branch gets a unique preview URL.

Vercel is created by the team behind Next.js and provides the best Next.js hosting experience, but works equally well with React, Vue, Svelte, Astro, and static HTML/CSS sites.

**How to sign up:** Go to https://vercel.com, sign up with GitHub (recommended), and click "New Project" to import your first repository. Your site will be live at `projectname.vercel.app` within minutes.

**Limitations:** The Hobby tier is for non-commercial personal use. Serverless Function execution is limited (but generous). Team features and commercial use require paid plans.

**Best free use case:** Deploy every class project to Vercel so you always have a live URL to include in your portfolio. The automatic preview deployments for every PR make it easy to demo features without affecting the production site.

**Upgrade path:** Vercel Pro ($20/month) adds team collaboration, commercial use, and more compute resources.

## 11. Netlify

**What is free:** 100GB bandwidth/month, 300 build minutes/month, unlimited personal projects.

Netlify is a strong alternative to Vercel for hosting static sites and JAMstack applications. The free tier is very generous for student use. Like Vercel, it connects to GitHub and deploys automatically on every push. Netlify Forms (for contact forms without backend code) is a free feature not available on Vercel's free tier.

Netlify's free tier also includes Netlify Identity (authentication), Edge Functions, and Netlify CMS integrations — making it excellent for slightly more complex projects.

**How to sign up:** Go to https://netlify.com, sign up with GitHub, and click "New site from Git". Select your repository and Netlify handles the rest.

**Limitations:** 300 build minutes/month is enough for most students but can run out on large projects. The `netlify.app` subdomain is provided on free tier; custom domains require configuration.

**Best free use case:** Use Netlify for static sites (plain HTML/CSS/JS), React apps that don't need server-side rendering, and projects that use Netlify Forms to handle contact form submissions without any backend code.

**Upgrade path:** Netlify Pro ($19/month) adds more build minutes, team features, and commercial use rights.

## 12. Replit

**What is free:** Browser-based development environment with free public Repls, limited compute.

Replit is a browser-based IDE that runs your code entirely in the cloud. You can write, run, and host code in 50+ languages without installing anything. Free public Repls are available without a subscription. Replit AI (Ghostwriter) is available with limits on free.

Replit is particularly popular for:
- Learning programming without setup friction
- Running quick experiments in new languages
- Building and hosting small apps and bots
- Real-time collaboration (multiple people coding together)

**How to sign up:** Go to https://replit.com and create a free account with Google, GitHub, or email. Start coding immediately in the browser.

**Limitations:** Free Repls are public (visible to everyone). Free compute is limited in RAM/CPU. Repls may "sleep" after inactivity and take a moment to wake up.

**Best free use case:** Use Replit for learning new programming concepts or languages where you don't want setup friction. Also great for workshop demos and teaching, where you want students to start coding in under 1 minute.

**Upgrade path:** Replit Core ($20/month) adds private Repls, more compute, and always-on hosting.

## 13. Lovable

**URL:** https://lovable.dev

**What is free:** Free messages to start (limited per month), generated projects can be exported.

Lovable is an AI full-stack app builder — you describe what you want in plain English and it builds a React + Supabase application for you. The free tier gives you a limited number of messages to generate and iterate on apps. Generated code is yours and can be exported to work with locally in VS Code or deploy elsewhere.

Lovable accelerates the prototype → MVP journey significantly. For student projects and hackathons, it can turn a 2-week project into a 2-hour project. The generated code is clean React and is educational to read.

**How to sign up:** Go to https://lovable.dev and sign up with GitHub. Start a new project by describing what you want to build.

**Limitations:** Free tier message limits. Complex custom features may require manual coding after generation. Lovable works best for standard CRUD apps and dashboards.

**Best free use case:** Use your free messages to build the initial prototype of a class project or hackathon idea. Export the generated code and then continue developing manually with AI assistance from Claude or ChatGPT.

**Upgrade path:** Lovable paid plans offer more messages per month.

## 14. Continue.dev

**What is free:** 100% free, completely open source. No subscription, no usage limits.

Continue.dev is an open-source AI coding assistant extension for VS Code and JetBrains. Unlike GitHub Copilot (which is a subscription service), Continue is free forever and works with any AI model you connect it to — including free models.

The recommended free setup: Connect Continue to Google AI Studio's Gemini 1.5 Flash API (free) for inline completions, and to Claude (free tier) or ChatGPT (free tier) for chat. This gives you a Copilot-like experience at zero cost.

**How to sign up:** Install the "Continue" extension from the VS Code marketplace. Follow the setup wizard to connect your preferred AI models. Documentation at https://continue.dev.

**Limitations:** Quality depends on which model you connect. Setup requires configuring API keys. Not as polished as GitHub Copilot in some edge cases.

**Best free use case:** If you haven't received the Student Developer Pack yet and want AI code completions today, Continue.dev + free Gemini API is the best zero-cost option. It genuinely rivals Copilot in day-to-day use.

**Upgrade path:** No upgrade — it's fully open source. Use better models (Claude Sonnet, GPT-4o) for better quality, at their respective costs.

## 15. Aider

**What is free:** The Aider tool itself is completely free and open source. You provide your own API key.

Aider is an open-source terminal AI coding assistant. Install it with `pip install aider-chat`, connect your API key, and you have a powerful coding agent that reads your codebase, makes changes, and commits them to Git. Aider supports OpenAI, Anthropic, Google Gemini, and local models (via Ollama).

The zero-cost setup: Install Aider + use the free Google Gemini API from AI Studio. This gives you a capable AI coding agent for free. The `--model gemini/gemini-1.5-flash` flag connects to Google's free tier.

**Installation:**
```bash
pip install aider-chat
export GOOGLE_API_KEY=your-free-key-from-aistudio
aider --model gemini/gemini-1.5-flash
```

**How to sign up:** Install with pip. Get a free API key from https://aistudio.google.com. No subscription needed.

**Limitations:** Command-line only — no GUI. Requires comfort with the terminal. API costs apply if you use paid models (OpenAI, Anthropic).

**Best free use case:** Use Aider with the free Gemini API for terminal-native AI coding. It's especially powerful for refactoring tasks and adding features to existing codebases, because it reads your entire project context before making changes.

**Upgrade path:** Use better models (Claude 3.5 Sonnet, GPT-4o) with their respective paid APIs for higher quality output.

## 16. Windsurf

**What is free:** Unlimited Codeium code completions (via free Codeium account), basic AI features.

Windsurf is an AI-first IDE from Codeium, built on VS Code. The key differentiator is that Windsurf includes Codeium's unlimited free code completions — no monthly limit. Codeium itself has been free for individual developers since its launch, funded by enterprise sales.

Windsurf also includes "Cascade" (their agentic AI feature), though advanced Cascade capabilities may require a paid plan. For students who want an AI IDE experience without the Cursor subscription, Windsurf is the best free option.

**How to sign up:** Go to https://codeium.com/windsurf, download Windsurf, and create a free Codeium account. Completions activate immediately.

**Limitations:** Cascade agent's most powerful features may require paid plan. Smaller community than VS Code/Cursor. Some VS Code extensions may have minor compatibility issues.

**Best free use case:** Use Windsurf as your daily driver IDE when you want AI completions without any monthly costs. The unlimited Codeium completions make it genuinely better than the free tier of Cursor for everyday use.

**Upgrade path:** Windsurf Pro provides access to more powerful models in Cascade.

## 17. Semgrep

**What is free:** Free for open source projects, community rules, and self-hosted use.

Semgrep is a static analysis tool that finds security vulnerabilities in code without running it. It has hundreds of community rules for detecting OWASP Top 10 vulnerabilities in Python, JavaScript, TypeScript, Java, Go, and more. The free tier includes all community rules and works on open-source projects without limits.

Install with `pip install semgrep` and run `semgrep --config=auto src/` to get an immediate security analysis of your project. This is invaluable for catching issues in AI-generated code.

**How to sign up:** Install via pip — no account required to use locally. For GitHub integration, sign up at https://semgrep.dev with GitHub.

**Limitations:** Open-source rules may miss novel vulnerabilities. The Pro/Team plans add proprietary rules and better integrations. Some false positives in complex codebases.

**Best free use case:** Add Semgrep to your GitHub Actions workflow so every push is automatically scanned for security issues. This is standard practice at professional software companies and takes less than 5 minutes to set up.

**Upgrade path:** Semgrep AppSec Platform (paid) adds proprietary rules, better reporting, and team features.

## 18. VS Code

**What is free:** Everything. VS Code is completely free and open source.

Visual Studio Code is arguably the most important free tool in this list. It's the most popular code editor in the world, supports every language through extensions, has a built-in terminal and debugger, and integrates with Git, GitHub, Docker, remote servers, and every AI tool covered in this guide. There is no paid version of VS Code — it's free forever.

VS Code's Extension Marketplace has 50,000+ extensions, the vast majority of which are free. The core features — syntax highlighting, IntelliSense, debugging, Git integration, integrated terminal — are all free and built-in.

**How to sign up:** Go to https://code.visualstudio.com and download. Available for Windows, Mac, and Linux. Also runs in the browser at https://vscode.dev.

**Limitations:** Not technically an AI tool on its own — AI features require extensions like GitHub Copilot or Continue.dev. Can be slow with many extensions enabled simultaneously.

**Best free use case:** Make VS Code your primary development environment. Learn its keyboard shortcuts (`Ctrl+Shift+P` for command palette, `Ctrl+`` ` for terminal), install extensions for your languages, and customize it to your workflow. It will serve you for your entire career.

**Upgrade path:** No upgrade — it's fully free. The paid AI extensions (GitHub Copilot) are separate purchases.

## 19. Git

**What is free:** Everything. Git is completely free and open source.

Git is the version control system that powers GitHub, GitLab, and Bitbucket. It's not an online service — it's software you install locally. Git tracks every change to your files, allows you to revert mistakes, and enables collaboration. It was created by Linus Torvalds (creator of Linux) and is used by virtually every software project in the world.

Git is the foundational skill for every software developer. Everything else in this guide builds on top of Git. Learning Git is non-negotiable for a career in software development.

**How to sign up:** Download Git from https://git-scm.com. On Mac, it's already installed (or install via Homebrew: `brew install git`). On Windows, download Git for Windows which includes Git Bash. On Linux: `sudo apt install git` or `sudo yum install git`.

**Limitations:** Command-line interface has a learning curve. Merge conflicts can be confusing for beginners. The mental model of branches, staging, and commits takes time to internalize.

**Best free use case:** Use Git from Day 1 of every project. Even personal projects benefit from version control — you can always go back to a working state if your changes break something. The most important commands to learn first: `git init`, `git add`, `git commit`, `git push`, `git pull`.

**Upgrade path:** No upgrade — it's free software. Host your repositories on GitHub (also free).
