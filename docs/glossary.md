# Glossary — Padang AI Coding Tools Class

| Term | Plain English Definition |
|------|--------------------------|
| **API** | A way for two programs to talk to each other. Like a waiter who takes your order to the kitchen and brings the food back. |
| **Artifact** (Claude) | A file or component Claude generates and renders in a live side panel — you can edit and iterate without copy-pasting. |
| **Branch** | A parallel version of a repository. Work on a feature branch without touching the stable `main` branch. |
| **CI/CD** | Continuous Integration / Continuous Deployment. Automated pipelines that test and deploy your code every time you push. |
| **CLI** | Command-Line Interface. A text-based way to run programs by typing commands in a terminal, rather than clicking. |
| **Commit** | A saved snapshot of your project at a specific moment, with a message describing what changed. |
| **Context Window** | The maximum amount of text an AI model can "see" at once — like short-term memory. Claude has 200,000 tokens. |
| **Deployment** | Making your application live and accessible to users on the internet. |
| **Environment Variable** | A value stored outside your code (in a `.env` file) so secrets like API keys are never committed to Git. |
| **Fork** | Your own copy of someone else's GitHub repository. Make changes without affecting the original. |
| **Hallucination** | When an AI confidently states something incorrect or makes up facts. Always verify AI output. |
| **Inference** | Running an AI model to generate a response. Each response is one "inference." |
| **LLM** | Large Language Model — the technology behind Claude, ChatGPT, and Gemini. Trained on text to predict the next token. |
| **MCP** | Model Context Protocol. An open standard that lets Claude Desktop connect to external tools — your filesystem, GitHub, browser, etc. |
| **Merge** | Combining changes from one branch into another (usually into `main`). |
| **Model** | The AI system trained to generate responses. Different models (Sonnet, Haiku, Opus) have different speed/capability trade-offs. |
| **PR (Pull Request)** | A request to merge changes from a branch into another. Creates a review thread for discussion. |
| **Prompt** | The text you send to an AI model. Better prompts produce better responses. |
| **Repo (Repository)** | A folder containing your project files plus the entire history of all changes. Hosted on GitHub. |
| **Secret** | Sensitive credentials — API keys, passwords, tokens — that must NEVER be committed to a public repository. |
| **Static Site** | A website made of plain HTML/CSS/JS files with no server-side processing. GitHub Pages hosts these for free. |
| **Terminal** | A text-based interface for running commands on your computer. Also called command line, shell, or CLI. |
| **Token** | The unit AI models use to measure text. One token ≈ 0.75 words. "Hello world" ≈ 2 tokens. |
| **Version Control** | Tracking every change to your files over time, so you can revert to any previous state. Git is the tool; GitHub is the host. |
