# Coding Prompts — Padang AI Coding Tools Class

Copy, fill in the blanks, paste into Claude or ChatGPT.

---

## Build from Scratch

```
You are a senior [React / Flask / Express / vanilla JS] developer.

Build a [component/feature name] that:
- Does [main behaviour]
- Has [specific feature 1]
- Has [specific feature 2]
- Uses [CSS framework or plain CSS]
- Does NOT use any external JS libraries

Return the complete code with brief inline comments.
```

---

## Add a Feature to Existing Code

```
Here is my existing [language] code:

[paste your code here]

Add [feature description].
Requirements:
- [requirement 1]
- [requirement 2]
- Do not break existing functionality

Return the complete updated file.
```

---

## Generate a Full HTML Page

```
Create a single complete HTML file for [page description].

Requirements:
- Responsive, mobile-first
- Semantic HTML5 elements
- [Plain CSS / Tailwind CDN / CSS variables]
- Sections: [list your sections]
- No JavaScript frameworks — vanilla JS only

Return the complete file starting with <!DOCTYPE html>.
```

---

## Write a Function with Tests

```
Write a [language] function called [name] that:
- Takes [inputs and types]
- Returns [output and description]
- Handles edge cases: [list edge cases]

Also write [N] unit tests using [Jest / pytest / Vitest].
Include the happy path, one edge case, and one error case.
```

---

## Refactor for Readability

```
Refactor this [language] code for readability and maintainability.
- Use descriptive variable names
- Extract repeated logic into helper functions
- Add JSDoc / docstring comments to public functions
- Do NOT change external behaviour or function signatures

[paste your code]
```
