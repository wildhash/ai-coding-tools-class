# Research & Learning Prompts
**Padang AI Coding Tools Class — examples/prompt-examples/**

---

## Comparing Technologies

```
Compare [Technology A] vs [Technology B] for [use case].

Structure your answer as:
1. One-paragraph summary of each
2. Comparison table: pros, cons, free tier, learning curve, community size
3. Clear recommendation for a student building [type of project]
4. One gotcha or pitfall beginners usually miss

Be direct — give me a recommendation, not just "it depends."
```

---

## Learning a New Concept

```
Explain [concept] to an Indonesian university student who knows basic Python
but has no experience with [related domain].

Use a real-world analogy relevant to everyday Indonesian life.
Then show a minimal code example (under 20 lines).
Then tell me: what is the most common mistake beginners make with this?
```

---

## Planning a Tech Stack

```
I want to build [project description].

My constraints:
- Skill level: [beginner/intermediate]
- Budget: free tier only (I'm a student in Indonesia)
- Timeline: [X weeks]
- Must work on: [web/mobile/both]

Recommend a tech stack. For each piece: what it is, why it fits my constraints,
and a link to the official docs. Keep the stack as small as possible.
```

---

## Summarising Documentation

```
Here is documentation for [library/API/tool]:

[paste docs text]

Give me:
1. What this thing does in 2 sentences
2. The 5 most important things to know before using it
3. A minimal working code example I can run immediately
4. The most common mistake people make when first using it
```

## Career Research

```
I am a [year] computer science student at a university in Padang, Indonesia.
I want to become a [role: frontend/backend/full-stack/ML/DevOps] developer.

For this role:
1. What are the 5 most in-demand skills right now (check current job postings)?
2. What 3 projects should I build for my portfolio to get my first job?
3. What free certifications or courses would an Indonesian employer recognise?
4. What is a realistic salary range for a junior [role] developer in Indonesia?
5. Which Indonesian companies hire junior [role] developers?
```

## Choosing Between Approaches

```
I am implementing [feature] and considering these approaches:

Option A: [describe]
- Pros: [list]
- Cons: [list]

Option B: [describe]
- Pros: [list]
- Cons: [list]

My constraints: [free tier, beginner-friendly, fast to implement, etc.]

Give me a direct recommendation with your reasoning.
Do not say "it depends" without following it with a specific recommendation.
```
