Student Projects

Five mini-projects to build during and after the workshop

1

Personal Profile Website

Beginner HTML/CSS/JS GitHub Pages Claude Web

Build a personal portfolio page and deploy it live to GitHub Pages. This is the classic first project that every developer can show to employers.

What to Build
  • Hero section with your name and tagline
  • About section with your background
  • Skills / tools you know
  • Projects section (can be this class!)
  • Contact section with links
Stretch Goals
  • Dark/light mode toggle
  • Smooth scroll navigation
  • CSS animations on scroll
  • Custom domain via Namecheap (free with Student Pack)
Starter Prompt 
Create a complete single-page personal portfolio website using HTML, CSS, and JavaScript.

My name is [Your Name]. I am studying [Your Major] at [Your University] in Padang, Indonesia. My interests include [interests].

Requirements:
- Sticky navigation bar with smooth scroll links
- Hero section with my name, tagline, and a "View My Work" button
- About section (2-3 sentences)
- Skills section showing: HTML, CSS, JavaScript, GitHub, Claude, VS Code
- Projects section with 2-3 placeholder project cards
- Contact section with GitHub and email links
- Responsive on mobile and desktop
- No frameworks — vanilla HTML/CSS/JS only
- Modern dark theme with purple accent colors (#7c3aed)

Return one complete HTML file starting with <!DOCTYPE html>.
2

AI Tools Directory

Beginner HTML/CSS/JS Cursor Copilot

Build a searchable directory of AI coding tools — your own reference page that you can customize and share with others.

What to Build
  • Grid of tool cards (name, description, free badge)
  • Filter by category (AI Chat, IDE, Review, Deploy)
  • Search bar that filters in real time
  • Each card links to the official tool URL
Stretch Goals
  • Add your own ratings or notes to each card
  • Save favorites to localStorage
  • Add a comparison mode (pick 2 tools, compare side by side)
Starter Prompt 
Build a searchable AI tools directory as a single HTML file.

Include these tools: Claude, ChatGPT, Cursor, VS Code, GitHub Copilot, CodeRabbit, Lovable, Windsurf, Continue.dev, Google AI Studio.

For each tool, show: emoji icon, name (as a hotlink to official URL), one-line description, category badge (AI Chat / IDE / Review / Deploy), free tier badge.

Features:
- CSS grid layout, 3 columns desktop / 2 tablet / 1 mobile
- Filter buttons at top: All, AI Chat, IDE, Review, Deploy
- Real-time search input that filters cards as you type
- Smooth hover animations on cards
- Dark theme matching: background #0f0f1a, accent #a78bfa

Return one complete HTML file with all CSS and JS inline.
3

Prompt Library Website

Intermediate JavaScript Claude Code GitHub Pages

Build a shareable library of your favorite AI prompts. A useful tool you'll actually keep using after the workshop.

What to Build
  • Categorized prompt cards (Coding, Debugging, Research, etc.)
  • One-click copy button on each prompt
  • Search by keyword
  • Fill-in-the-blank placeholders highlighted
Stretch Goals
  • Add your own prompts via a form (stored in localStorage)
  • Export all prompts as a markdown file
  • Rate prompts (helpful / not helpful)
Starter Prompt 
Build a prompt library website as a single HTML file.

Include 3 categories, 3 prompts each (9 total):

Coding: (1) Write a function, (2) Refactor for readability, (3) Add error handling
Debugging: (1) Fix an error, (2) Explain unexpected output, (3) Debug async code
Learning: (1) Explain a concept, (2) Compare two approaches, (3) Suggest a learning plan

Each prompt card should have:
- Category badge
- Prompt title
- Full prompt text (use [PLACEHOLDERS] in brackets for fill-in parts)
- Copy button that copies the text and shows "Copied!"

Features:
- Filter by category
- Keyword search
- Dark theme, responsive
- Highlight [PLACEHOLDER] text in a different color

Return one complete HTML file.
4

JavaScript Task Tracker

Intermediate JavaScript Copilot CodeRabbit

Build a functional task tracker app using Copilot for autocomplete and CodeRabbit to review your PR. Great for learning JavaScript state management.

What to Build
  • Add tasks with a title and optional due date
  • Mark tasks complete (strikethrough)
  • Delete tasks
  • Persist tasks in localStorage
  • Filter: All / Active / Completed
Stretch Goals
  • Drag-to-reorder tasks
  • Priority levels (High / Medium / Low) with color coding
  • Export tasks as CSV
  • Open a PR and let CodeRabbit review your JS
Starter Prompt 
Build a task tracker app as a single HTML file using vanilla JavaScript.

Features:
- Input field + "Add Task" button
- Task list with: checkbox (toggle complete), task text, delete button
- Completed tasks shown with strikethrough and reduced opacity
- Three filter buttons: All, Active, Completed
- Task count: "X tasks remaining"
- All tasks persist in localStorage (survive page refresh)
- Empty state message when list is empty

Design:
- Dark theme, clean card layout
- Smooth animations when adding/removing tasks
- Responsive on mobile
- No external libraries or frameworks

Include input validation: prevent empty tasks.
Return one complete HTML file.

Beginner Portfolio Page

Intermediate HTML/CSS/JS Claude Code All Tools

Your class portfolio — document what you built during the workshop, what tools you used, and what you learned. Use every tool from the class to build it.

What to Build
  • About: who you are, where you study
  • Workshop summary: what you learned today
  • Tools section: which ones you liked most
  • Projects gallery: link to everything you built
  • Next steps: what you plan to build next
Stretch Goals
  • Built with Claude Code from the terminal
  • CodeRabbit review on the final PR
  • Bilingual EN/ID toggle (like this site!)
  • Custom domain from Student Pack
Claude Code Starter Session 
mkdir my-workshop-portfolio && cd my-workshop-portfolio
git init
claude

# Then in the Claude Code session:
/init

Build a personal workshop portfolio website called "[Your Name]'s AI Coding Journey".

Content:
- Hero: my name, "I attended the Padang AI Coding Tools Class at IndigoSpace"
- What I Learned: list of 5 things I learned today
- Tools I Used: cards for Claude, ChatGPT, VS Code, Cursor, GitHub, CodeRabbit
- What I Built: 2-3 project cards with placeholders
- Next Steps: what I want to build next
- Footer with my GitHub link

Design: dark theme, purple accents, responsive
Files: index.html + style.css + script.js (separate files)

After building: git add . && git commit -m "feat: initial workshop portfolio"
